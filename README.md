# DevOps Python & AWS Automation

Python scripts for AWS infrastructure automation using Boto3.

## Scripts

| Script | Purpose |
|--------|---------|
| `first-python.py` | Python basics - variables, lists, dictionaries, loops |
| `functions-demo.py` | Functions and error handling (try/except) |
| `aws-python-test.py` | Connect to AWS - EC2 and S3 basic queries |
| `aws-manager.py` | Full AWS resource audit - EC2, S3, IAM, Security Groups |

## Usage

```bash
python3 aws-manager.py
python3 aws-manager.py | grep "viewer"
```
